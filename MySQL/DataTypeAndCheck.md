# 数据类型和约束

## 一、常见的数据类型

### 1. 概述

+ **数值型**
  + 整型
  + 小数
    + 定点数
    + 浮点数
+ **字符型**
  + 较短的文本
    + char，varchar
  + 较长的文本
    + text，blob（较长的二进制数据）
+ **日期型**

    |  Mediumint   |   3    |
    | Int、Integer |   4    |
    |    Bigint    |   8    |
